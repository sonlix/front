## Получение всех тегов авторизованного пользователя

#### Описание:
Запрос присылает все имеющиеся теги, даже те которые не используются.

#### Роут:
`GET /ajax/users/tags`

#### Пример:
Request
```
GET /ajax/users/tags
```
Response
```
Status:
200 OK

body:
["tag1","tag2","tag3"]
```
